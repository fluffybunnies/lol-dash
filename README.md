# lol-dash
League of Legends Dashboard


## Build
```bash
npm install
```


## Run
```bash
npm start
```


## Keep Up To Date
- Grab the latest champ file here: http://ddragon.leagueoflegends.com/cdn/10.5.1/data/en_US/champion.json
